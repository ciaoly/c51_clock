**本项目纯属应付课设.** 

### 一个 51 单片机 + DS1302 实现的数字时钟, 支持闹钟; 整点报时; 12/24进制转换.  

#### 使用方法  
##### 普通状态下:  
* 按下第一颗按钮进入 "设置" 状态;  
* 按下第二颗按钮切换 12/24 进制;  
* 按下第三颗按钮静音(使闹钟停止发声);  
* 按下第四颗按钮进入 "闹钟设置" 状态;  

##### 设置状态下:  
* 按下第一颗按钮切换待设置对象;  
* 按下第二颗按钮向上加一(支持溢出检测);  
* 按下第三颗按钮向下减一(支持溢出检测);  
* 第四颗按钮无效;  

##### 闹钟设置状态下:
* 第一颗按钮无效  
* 按下第二颗按钮向上加一(支持溢出检测);  
* 按下第三颗按钮向下减一(支持溢出检测);  
* 按下第四颗按钮切换待设置项目(依次为: 关闭闹钟/ 设置"时"/ 设置"分")
* 长按第四颗按钮保存闹钟设置
